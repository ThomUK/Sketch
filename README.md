# Sketch

<!-- badges: start -->

<!-- badges: end -->

Sketch is a simple R Shiny drawing app, a lightweight way to make and share a sketch online. Sketches can either be saved to the clipboard (e.g. for pasting into a meeting chat), or saved as a png file for onward sharing.

## Features: 
* Pen colours (Black, Grey, Blue, Red, Amber, Green) 
* Pen size can be varied 
* Optionally, a title can be added to the image

## Tech details

There is no server-side element to the app. All functionality is rendered client-side via javascript.
